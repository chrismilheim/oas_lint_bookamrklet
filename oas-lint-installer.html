🔍 LintErrors, warnings, suggestions — collapsible sections. operationIds, responses, parameters, security, schemas.

⚡ 60KBNo external dependencies. No iframes. No new tabs. Works instantly, fully within GitHub's CSP.
Why no Swagger UI: GitHub's CSP blocks eval(), new Function(), all iframe variants (srcdoc, blob:, data:), and external scripts. Swagger UI requires eval to work. So this bookmarklet builds its own spec explorer from scratch using pure DOM — parameters, types, response codes, request bodies, tag grouping — all rendered inline with zero dependencies beyond js-yaml.
